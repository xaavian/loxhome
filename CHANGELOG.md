# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2025-02-17

### Added
- Initial release
- Loxone-inspired dark UI with tile-based dashboard
- 4-tab navigation: Favourites, Central, Rooms, Categories
- Loxone-style Room & Category selectors with icon cards and drill-down
- Real-time entity updates via Home Assistant WebSocket API
- Support for 13 entity domains: light, switch, climate, cover, media_player, sensor, camera, lock, alarm_control_panel, fan, input_boolean, automation
- Drag & drop tile editing with SortableJS
- Demo mode with ~35 simulated entities
- Persistent config via API (file) or localStorage fallback
- Home Assistant add-on with Ingress support
- Responsive design for tablets, phones, and desktops
